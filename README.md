# WAF_PASS

WAF_PASS 是一个用于规避 Web 应用防火墙（WAF）检测的字符串拼接工具。通过将字符串分割成多个小片段并使用 `+` 运算符拼接，可以有效地绕过 WAF 的检测机制。

## 功能特性

- **字符串分割**：将输入的字符串按照指定长度分割成多个片段
- **符号处理**：可以指定需要单独拼接的符号
- **去重功能**：可以去除重复的拼接符号
- **实时转换**：输入后立即生成转换结果
- **用户友好界面**：简洁直观的网页界面，易于使用

## 技术实现

- **前端技术**：纯 HTML、CSS 和 JavaScript 实现，无需后端服务器
- **响应式设计**：适配不同屏幕尺寸
- **实时处理**：在浏览器中本地处理，无需网络请求

## 使用方法

1. 打开 `index.html` 文件
2. 在 "输入字符串" 文本框中输入需要转换的字符串
3. 在 "拼接长度" 中设置每个拼接段的字母数量（默认为 2）
4. 在 "需要拼接的符号" 中输入需要单独拼接的符号，多个符号用空格分隔
5. 选择是否 "去除重复的拼接符号"
6. 点击 "转换" 按钮
7. 转换结果会显示在 "转换结果" 文本框中

## 示例

### 输入：
```
admin' OR 1=1 --
```

### 输出（拼接长度为 2）：
```
'ad'+'mi'+'n' 'OR' '1'='1' '--'
```
